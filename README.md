# Number Guessing Game
Project Overview: 
The Number Guessing Game is a simple Java console application that allows the user to guess a randomly generated number between 1 and 100. The main objective of the game is to correctly guess the hidden number. After each guess the program provides feedback to let the user know whether the guess is too high, too low or correct. The game continues until the correct number is guessed.

This project was developed to demonstrate the use of Java programming concepts such as object-oriented programming (OOP), methods, loops, conditional statements, random number generation and user input validation. The project also focuses on writing clean, readable and well-documented code that follows good programming practices.

Features:
Generates a random number between 1 and 100.
Accepts user input from the keyboard.
Validates user input to ensure only numbers are accepted.
Gives feedback after every guess.
Counts and displays the total number of attempts made by the player.


Technologies Used:
Java
Java Development Kit (JDK)
IntelliJ IDEA

How to Run the Program:
Open the project in your IntelliJ Java IDE.
Compile the Java files.
Run the Main.java class which is NumberGuessingGame.
Follow the instructions displayed on the console.
Continue guessing until you find the correct number.

Project Structure:
Game.Java - Contains the game logic, including random number generation, checking guesses and tracking the number of attempts.
Player.java – Contains Players name and the total score of the player.
NumberGuessingGame.java – Contains the main method that starts the game and handles user interaction.

Learning Outcomes:
Developing this project helped strengthen my understanding of Java programming and object-oriented programming principles. It also improved my ability to work with classes, methods, loops, conditional statements and exception handling through input validation. Additionally the project reinforced the importance of writing organised, well-commented code that is easy to understand and maintain.

Conclusion:
The Number Guessing Game is a simple but effective project that demonstrates the practical application of fundamental Java programming concepts. Although the game is straightforward it provides a solid foundation for building more advanced applications. Future improvements could include adding difficulty levels, limiting the number of attempts, developing a graphical user interface (GUI) to make the game more interactive.
